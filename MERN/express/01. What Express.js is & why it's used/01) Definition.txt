# What is Express.js and Why is it Used?

## What is Express.js?

Express.js is a lightweight and flexible web application framework built on top of Node.js. It helps developers build web servers, APIs, and backend applications quickly and easily.

Normally, creating a server using only Node.js requires writing a lot of code manually. Express.js simplifies this process by providing ready-made tools and features.

It is one of the most popular backend frameworks used in MERN stack development.

---

# Why is Express.js Used?

Express.js is mainly used to:

 Build REST APIs
 Handle HTTP requests and responses
 Create backend servers
 Connect frontend with databases
 Manage routes and middleware
 Build scalable web applications

---

# Example Without Express.js

Using only Node.js:

```js
import http from "http";

const server = http.createServer((req, res) => {
  if (req.url === "/") {
    res.end("Home Page");
  }
});

server.listen(5000);
```

This becomes complicated as the project grows.

---

# Example With Express.js

```js
import express from "express";

const app = express();

app.get("/", (req, res) => {
  res.send("Home Page");
});

app.listen(5000, () => {
  console.log("Server running");
});
```

This is cleaner, shorter, and easier to manage.

---

# Main Features of Express.js

## 1. Routing

Routing means handling different URLs.

Example:

```js
app.get("/users", (req, res) => {
  res.send("Users Page");
});

app.post("/login", (req, res) => {
  res.send("Login Successful");
});
```

---

## 2. Middleware Support

Middleware functions run before the final response.

Used for:

 Authentication
 Logging
 Error handling
 Validation

Example:

```js
app.use(express.json());
```

This middleware converts incoming JSON data into JavaScript objects.

---

## 3. REST API Development

Express is widely used for creating APIs.

Example:

```js
app.get("/products", (req, res) => {
  res.json([
    { id: 1, name: "Laptop" },
    { id: 2, name: "Phone" }
  ]);
});
```

---

## 4. Fast Development

Express removes boilerplate code and speeds up backend development.

---

## 5. Easy Database Integration

Works easily with databases like:

 MongoDB
 MySQL
 PostgreSQL
