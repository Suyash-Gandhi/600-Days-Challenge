## 13. Wallets and Address Generation


# 🔐 1. What is a Wallet?

A blockchain wallet does NOT actually store your cryptocurrency.

👉 Instead, it stores:

 Private keys
 Public keys
 Helps you interact with the blockchain

### 📌 Key Idea:

> Coins are stored on the blockchain, not in the wallet.
> The wallet just gives you access control.

---

# 🔑 2. Key Components of a Wallet

### 1. Private Key

 A random 256-bit number
 Example:

  ```
  5KQwrPbwdL6PhXujxW37FSSQJkz5... (shortened)
  ```
 Used to:

   Sign transactions
   Prove ownership

⚠️ If someone gets your private key → they control your funds

---

### 2. Public Key

 Derived from private key using elliptic curve cryptography
 Safe to share

---

### 3. Address

 Derived from public key
 Used to receive funds

Example (Bitcoin):

```
1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa
```

---

# 🔄 3. Address Generation Process (Step-by-Step)

This is VERY IMPORTANT for interviews + deep understanding

### Step 1: Generate Private Key

 Random 256-bit number

### Step 2: Generate Public Key

 Using Elliptic Curve Cryptography
 Bitcoin uses curve:

   secp256k1

---

### Step 3: Hash the Public Key

Two hashing steps:

1. SHA-256
2. RIPEMD-160

 SHA-256:

   SHA-256
 RIPEMD-160:

   RIPEMD-160

👉 Result = Public Key Hash

---

### Step 4: Add Version + Checksum

 Add network version (mainnet/testnet)
 Add checksum (error detection)

---

### Step 5: Encode Address

 Using:

   Base58Check

👉 Final Output = Wallet Address

---

# 🧠 Summary Flow

```
Private Key
   ↓ (ECC)
Public Key
   ↓ (SHA-256 → RIPEMD-160)
Public Key Hash
   ↓ (Version + Checksum)
Encoded Address (Base58)
```

---

# 🪪 4. Types of Wallets

## 🔥 1. Hot Wallets (Online)

 Connected to internet
 Examples:

   MetaMask
   Trust Wallet

### Pros:

 Easy to use
 Fast transactions

### Cons:

 More vulnerable to hacks

---

## ❄️ 2. Cold Wallets (Offline)

 Not connected to internet

Examples:

 Ledger Nano S
 Trezor

### Pros:

 Highly secure

### Cons:

 Less convenient

---

## 🧾 3. Custodial vs Non-Custodial

### Custodial Wallet

 Third party controls keys
 Example: Exchanges

👉 You don’t own your crypto fully

---

### Non-Custodial Wallet

 You control private keys
 Example:

   MetaMask

👉 Full ownership

---

# 🧬 5. HD Wallets (Hierarchical Deterministic)

Modern wallets use one seed phrase to generate many addresses.

### Example:

 12/24 word seed phrase:

```
apple river moon ...
```

Standards:

 BIP-32
 BIP-39

### Benefits:

 Backup once → recover all addresses
 Generate infinite addresses
