
## 1. What is a hash function? (Plain English)

A hash function is a special function that:

 Takes any input (text, number, file, transaction data, anything)
 Produces a fixed-length output called a hash
 The same input always gives the same output
 Even a tiny change in input gives a completely different output

 Input: `hello` (small change)
 Hash: `aaf4c61ddcc5e8a2dabede0f3b482cd9...`
  Completely different.

## 6. SHA-256 in Proof of Work (Bitcoin example)
