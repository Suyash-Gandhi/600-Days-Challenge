## 6. Security properties of Keccak

Keccak provides all essential cryptographic guarantees:

1. Deterministic

    Same input â†’ same output

2. Collision resistance

    Extremely hard to find two inputs with same hash

3. Pre-image resistance

    Cannot reverse hash to original data

4. Avalanche effect

    Small input change â†’ massive output change

These properties are critical for blockchain security.
