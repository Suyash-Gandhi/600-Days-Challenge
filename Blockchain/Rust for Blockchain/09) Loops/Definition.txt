## ğŸ§® 3. The `while` Loop

### Example:

```rust

    let mut number = 3;

    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }

    println!("Liftoff!");
}
```

### ğŸ§© Explanation:

## ğŸ§± 5. `break` and `continue` in Loops

| Keyword    | Description                                       |
| ---------- | ------------------------------------------------- |
| `break`    | Exits the loop immediately                        |
| `continue` | Skips the current iteration and moves to the next |

### Example:

```rust

        if i == 5 {
            continue; // Skip number 5
        }
        if i == 8 {
            break; // Stop at number 8
        }
        println!("{}", i);
    }
}
```

```rust
        for j in 1..=3 {

                break 'outer; // breaks outer loop
            }
            println!("i = {}, j = {}", i, j);
        }
    }
}
```
