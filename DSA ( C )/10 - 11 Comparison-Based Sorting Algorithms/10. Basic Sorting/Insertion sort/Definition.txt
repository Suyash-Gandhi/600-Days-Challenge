

### 1. What is Insertion Sort?

Insertion Sort is a comparison-based sorting algorithm that works the same way you sort playing cards in your hand.

 It builds the sorted array one element at a time
 Each new element is inserted into its correct position in the already sorted part

---

### 2. Core Idea (Very Important)

### 10. When Should You Use Insertion Sort?

Use Insertion Sort when:

 Input size is small
 Data is almost sorted
 You need a stable algorithm
 Memory usage must be minimal

Avoid it when:

 Dataset is large
 Performance is critical
