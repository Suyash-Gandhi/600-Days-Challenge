# 🟢 4. Operators in Python

Operators are symbols or keywords used to perform operations on variables and values.

👉 Example:

```python
a = 5
b = 3
print(a + b)  # 8
```

Here, `+` is an **operator**, and `a`, `b` are **operands**.

---

# 🔹 Types of Operators in Python

Python has **7 main types of operators**:

1. Arithmetic Operators
2. Comparison (Relational) Operators
3. Logical Operators
4. Assignment Operators
5. Membership Operators
6. Identity Operators
7. Bitwise Operators

---

# 🟡 1. Arithmetic Operators

Used for **mathematical calculations**.

### Operators List

| Operator | Meaning             | Example       |
| -------- | ------------------- | ------------- |
| `+`      | Addition            | `5 + 2 = 7`   |
| `-`      | Subtraction         | `5 - 2 = 3`   |
| `*`      | Multiplication      | `5 * 2 = 10`  |
| `/`      | Division (float)    | `5 / 2 = 2.5` |
| `//`     | Floor division      | `5 // 2 = 2`  |
| `%`      | Modulus (remainder) | `5 % 2 = 1`   |
| `**`     | Power               | `2 ** 3 = 8`  |

---

### 📌 Important Concepts

#### 1. Division Types

```python
print(5 / 2)   # 2.5  (float)
print(5 // 2)  # 2    (integer)
```

#### 2. Modulus Use Case

```python
# Check even or odd
num = 4
print(num % 2 == 0)  # True (even)
```

#### 3. Exponentiation

```python
print(3 ** 2)  # 9
```

---

### 🧠 Best Practice

* Use `//` when you need **integer result**
* Use `%` for **cyclic logic (even/odd, loops, patterns)**

---

# 🟡 2. Comparison Operators

Used to **compare values** → result is always `True` or `False`.

### Operators List

| Operator | Meaning          |
| -------- | ---------------- |
| `==`     | Equal            |
| `!=`     | Not equal        |
| `>`      | Greater than     |
| `<`      | Less than        |
| `>=`     | Greater or equal |
| `<=`     | Less or equal    |

---

### 📌 Examples

```python
a = 10
b = 5

print(a > b)   # True
print(a == b)  # False
print(a != b)  # True
```

---

### 📌 Chaining Comparisons (Advanced & Important)

```python
x = 10
print(5 < x < 15)  # True
```

👉 Python evaluates this as:

```python
(5 < x) and (x < 15)
```

---

### 🧠 Best Practice

* Use comparison operators in:

  * `if` statements
  * loops
  * filtering data

---

# 🟡 3. Logical Operators

Used to **combine multiple conditions**.

### Operators

| Operator | Meaning                      |
| -------- | ---------------------------- |
| `and`    | Both conditions must be True |
| `or`     | At least one must be True    |
| `not`    | Reverses result              |

---

### 📌 Examples

```python
a = 10

print(a > 5 and a < 20)  # True
print(a > 5 or a < 5)    # True
print(not(a > 5))        # False
```

---

### 📌 Truth Table

#### AND

| A | B | Result |
| - | - | ------ |
| T | T | T      |
| T | F | F      |
| F | T | F      |
| F | F | F      |

#### OR

| A | B | Result |
| - | - | ------ |
| T | T | T      |
| T | F | T      |
| F | T | T      |
| F | F | F      |

---

### ⚠️ Short-Circuiting (Important Concept)

```python
print(False and print("Hello"))  # nothing prints
print(True or print("Hello"))    # nothing prints
```

👉 Python **stops evaluation early** when result is already known.

---

### 🧠 Best Practice

* Use `and` for **range checks**
* Use `or` for **multiple options**
* Use `not` carefully (can reduce readability)

---

# 🟡 4. Assignment Operators

Used to **assign values to variables**.

---

### Basic Assignment

```python
x = 5
```

---

### Compound Assignment Operators

| Operator | Meaning             | Example   |
| -------- | ------------------- | --------- |
| `+=`     | Add and assign      | `x += 3`  |
| `-=`     | Subtract and assign | `x -= 3`  |
| `*=`     | Multiply and assign | `x *= 3`  |
| `/=`     | Divide and assign   | `x /= 3`  |
| `//=`    | Floor divide assign | `x //= 3` |
| `%=`     | Modulus assign      | `x %= 3`  |
| `**=`    | Power assign        | `x **= 2` |

---

### 📌 Example

```python
x = 10
x += 5   # x = x + 5
print(x) # 15
```

---

### 🧠 Best Practice

* Use compound operators for **clean & concise code**
* Avoid overusing in complex expressions

---

# 🟡 5. Membership Operators

Used to check if a value exists in a sequence.

---

### Operators

| Operator | Meaning              |
| -------- | -------------------- |
| `in`     | Value exists         |
| `not in` | Value does not exist |

---

### 📌 Examples

```python
my_list = [1, 2, 3, 4]

print(2 in my_list)      # True
print(5 not in my_list)  # True
```

---

### 📌 String Example

```python
text = "hello"

print("h" in text)   # True
print("z" in text)   # False
```

---

### 🧠 Best Practice

* Use `in` for:

  * Lists
  * Strings
  * Dictionaries (checks keys)

```python
my_dict = {"a": 1}
print("a" in my_dict)  # True
```

---

# 🟡 6. Identity Operators

Used to check if two variables refer to the **same memory object**.

---

### Operators

| Operator | Meaning          |
| -------- | ---------------- |
| `is`     | Same object      |
| `is not` | Different object |

---

### 📌 Example

```python
a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a is b)  # True (same object)
print(a is c)  # False (different objects)
```

---

### 📌 Difference: `==` vs `is`

```python
print(a == c)  # True (values same)
print(a is c)  # False (different memory)
```

---

### 🧠 Best Practice

* Use `==` for value comparison
* Use `is` for:

  * checking `None`

```python
if x is None:
    print("No value")
```

---

# 🟡 7. Bitwise Operators (Basics)

Used to perform operations at **binary level (bits)**.

---

### Operators

| Operator | Meaning     |    |
| -------- | ----------- | -- |
| `&`      | AND         |    |
| `        | `           | OR |
| `^`      | XOR         |    |
| `~`      | NOT         |    |
| `<<`     | Left shift  |    |
| `>>`     | Right shift |    |

---

### 📌 Binary Understanding

Example:

```python
5 = 101
3 = 011
```

---

### 📌 Examples

#### AND (`&`)

```python
print(5 & 3)  # 1 (001)
```

#### OR (`|`)

```python
print(5 | 3)  # 7 (111)
```

#### XOR (`^`)

```python
print(5 ^ 3)  # 6 (110)
```

#### NOT (`~`)

```python
print(~5)  # -6
```

---

### 📌 Shift Operators

#### Left Shift

```python
print(5 << 1)  # 10
```

#### Right Shift

```python
print(5 >> 1)  # 2
```

---

### 🧠 When Used?

* Rare in beginner code
* Used in:

  * low-level programming
  * optimizations
  * cryptography

# 🔥 Operator Precedence (Very Important)

Order in which operations are evaluated.

### Priority (High → Low)

1. `**`
2. `* / // %`
3. `+ -`
4. Comparison (`==`, `>`, etc.)
5. `not`
6. `and`
7. `or`

---

### 📌 Example

```python
result = 2 + 3 * 4
print(result)  # 14
```

👉 Because `*` has higher precedence than `+`

---

### 📌 Use Parentheses (Best Practice)

```python
result = (2 + 3) * 4  # 20
```
