Let’s break down Dictionaries & their methods in a way that is clear, practical, and deeply understandable (like you’ll actually use them in projects).

# 🟢 7. Dictionaries in Python

## 🔹 1. What is a Dictionary?

A dictionary is a collection of key-value pairs.

👉 Think of it like a real-world dictionary:

 Word → Meaning
 Key → Value

### Example:

```python
student = {
    "name": "Suyash",
    "age": 21,
    "course": "B.Tech"
}
```

### 🔑 Key Idea:

 `"name"` → key
 `"Suyash"` → value

---

## 🔹 2. Key Characteristics

### 1. Unordered (before Python 3.7) / Ordered (after 3.7)

 Now dictionaries preserve insertion order

### 2. Keys must be:

 Unique
 Immutable (string, number, tuple)

❌ Invalid:

```python
{ ["a"]: 1 }  # list can't be key
```

### 3. Values can be anything:

 string, number, list, even another dictionary

---

## 🔹 3. Creating Dictionaries

### Method 1: Using {}

```python
person = {"name": "Aman", "age": 22}
```

### Method 2: Using dict()

```python
person = dict(name="Aman", age=22)
```

### Method 3: Empty dictionary

```python
data = {}
```

---

## 🔹 4. Accessing Values

### Using key:

```python
student["name"]
```

⚠️ Error if key doesn't exist

---

### Using `.get()` (safe way)

```python
student.get("name")
student.get("marks", 0)  # default value
```

👉 Best practice: Use `.get()` when unsure if key exists

---

## 🔹 5. Adding & Updating Values

### Add new key:

```python
student["marks"] = 95
```

### Update existing:

```python
student["age"] = 22
```

---

## 🔹 6. Removing Elements

### 1. `pop()`

```python
student.pop("age")
```

### 2. `del`

```python
del student["name"]
```

### 3. `clear()`

```python
student.clear()
```

---

# 🟡 7. Important Dictionary Methods

---

## 🔸 1. `keys()`

Returns all keys

```python
student.keys()
```

👉 Output:

```
dict_keys(['name', 'age'])
```

---

## 🔸 2. `values()`

Returns all values

```python
student.values()
```

---

## 🔸 3. `items()`

Returns key-value pairs

```python
student.items()
```

👉 Output:

```
dict_items([('name', 'Suyash'), ('age', 21)])
```

---

## 🔸 4. `update()`

Adds or updates multiple values

```python
student.update({"age": 23, "city": "Delhi"})
```

---

## 🔸 5. `popitem()`

Removes last inserted item

```python
student.popitem()
```

---

## 🔸 6. `setdefault()`

Gets value or creates key if not exists

```python
student.setdefault("marks", 0)
```

👉 Useful for counters / defaults

---

## 🔸 7. `copy()`

Creates a shallow copy

```python
new_student = student.copy()
```

---

# 🔵 8. Looping Through Dictionary

---

### Loop through keys:

```python
for key in student:
    print(key)
```

---

### Loop through values:

```python
for value in student.values():
    print(value)
```

---

### Loop through key-value:

```python
for key, value in student.items():
    print(key, value)
```

---

# 🔵 9. Nested Dictionaries

```python
students = {
    "student1": {"name": "Aman", "age": 20},
    "student2": {"name": "Suyash", "age": 21}
}
```

Access:

```python
students["student1"]["name"]
```

---

# 🔵 10. Dictionary vs List (Important)

| Feature   | Dictionary    | List           |
| --------- | ------------- | -------------- |
| Access    | Key-based     | Index-based    |
| Speed     | Faster lookup | Slower search  |
| Structure | Key-value     | Ordered values |

# 🔵 11. Real-World Use Cases

---

## 🔹 1. JSON Data (VERY IMPORTANT)

```python
user = {
    "id": 1,
    "email": "test@gmail.com"
}
```

👉 Used in APIs, backend, MongoDB

---

## 🔹 2. Counting frequency

```python
word = "apple"
freq = {}

for char in word:
    freq[char] = freq.get(char, 0) + 1
```

---

## 🔹 3. Storing configurations

```python
config = {
    "theme": "dark",
    "language": "en"
}
```

# 🔴 12. Common Mistakes

---

### ❌ Accessing missing key

```python
student["marks"]  # Error
```

✔️ Fix:

```python
student.get("marks")
```

---

### ❌ Duplicate keys

```python
{"a": 1, "a": 2}  # keeps only last
```

---

### ❌ Confusing list with dict

```python
data[0]  # works only for list, not dict
```
