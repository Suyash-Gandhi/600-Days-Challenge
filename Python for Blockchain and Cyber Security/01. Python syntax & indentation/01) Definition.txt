

## 🔹 1. What is Python Syntax?

Syntax = rules for writing code correctly

In Python, syntax is clean, readable, and close to English.

### Example:

```python
print("Hello World")
```

👉 No semicolon `;` needed
👉 No brackets required for blocks

---

## 🔹 2. What is Indentation?

Indentation = spaces at the beginning of a line

Python uses indentation to define blocks of code (like inside if, loops, functions).

---

## 🔹 3. Why Indentation Matters (VERY IMPORTANT)

In other languages:

```js
if (true) {
    console.log("Hello");
}
```

In Python:

```python
if True:
    print("Hello")
```

👉 Instead of `{}`, Python uses indentation (spaces)

---

## 🔹 4. Correct vs Incorrect Indentation

### ✅ Correct:

```python
if True:
    print("Hello")
```

### ❌ Incorrect:

```python
if True:
print("Hello")   # Error
```

👉 Python will throw:

```
IndentationError: expected an indented block
```

---

## 🔹 5. Standard Indentation Rule

 Use 4 spaces (recommended)
 Don’t mix tabs and spaces

```python
if True:
    print("Line 1")
    print("Line 2")
```

---

## 🔹 6. Nested Indentation (Important)

```python
if True:
    print("Level 1")

    if True:
        print("Level 2")
```

👉 Each level = deeper block

---

## 🔹 7. Where Indentation is Used

You’ll use indentation in:

 `if / else`
 `for loops`
 `while loops`
 `functions`
 `classes`

Example:

```python
for i in range(3):
    print(i)
```

---

## 🔹 8. Common Beginner Mistakes

 ❌ Forgetting indentation after `:`
 ❌ Mixing tabs and spaces
 ❌ Wrong alignment
